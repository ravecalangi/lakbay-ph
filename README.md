# Lakbay — Travel & Weather Explorer

## Description
Lakbay is a modern travel and weather web application built for Filipino travelers. It allows users to explore tourist destinations, view real-time weather data, check air quality, and interact with an AI travel assistant in one platform.

## Features
- Search tourist destinations by Philippine city
- Real-time weather information using OpenWeather API
- 5-day weather forecast
- Air Quality Index (AQI) monitoring
- Live sunrise and sunset visualization
- Tourist spot explorer with categories and filters
- AI-powered travel assistant chatbot
- Responsive and modern UI design
- Dark mode support
- Interactive charts for temperature, rain, and wind data
- Automatic geolocation detection
- Dynamic tourist place images from Wikipedia and Wikimedia Commons

## Technologies Used
- JavaScript (ES6)
- HTML5
- CSS3
- OpenWeather API
- OpenTripMap API
- OpenRouter API
- Chart.js
- Font Awesome
- Google Fonts

## How to Run
1. Download or clone the repository.
2. Make sure the project structure contains:
   - landing-page
   - main-page
   - HTML, CSS, and JavaScript files
3. Open `index.html` in your browser.
4. Allow location access for automatic city detection (optional).

## Usage
1. Open the application in your browser.
2. Search for a Philippine city using the search bar.
3. Explore tourist destinations and weather information.
4. Switch between Explore and Weather tabs.
5. Use the AI Assistant for travel-related questions.

### Example

Input:
Manila

Output:
- Current temperature and weather condition
- Air quality information
- 5-day weather forecast
- Tourist spots around Manila
- AI travel recommendations
